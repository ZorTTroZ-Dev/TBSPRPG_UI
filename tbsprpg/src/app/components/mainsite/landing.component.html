Text-based single-player role-playing game

Welcome to TbspRpg, where you'll experience epic adventure from the comfort of your web browser. Currently we can use the site to create basic branching stories, and we're working hard to add exciting features. Features such as, tactical combat, interesting choices that can impact the story, deep role-playing, and offline play. Register to stay updated!

Features

Browser Based

Experience adventure where ever you have a browser. Your game is saved, so you can pick up where you left off from any device.

Deep Role-Playing (Eventually...)

Explore deep complex worlds! Since the games are text based we'll be able to quickly create reactive and interesting experiences. (Eventually...)

Create Your Own (Eventually...)

Once the web interface is finalized and more features have been completed, we'll allow users to create their own adventures to share with the world.
